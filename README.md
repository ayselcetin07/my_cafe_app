# Kafe Sipariş Uygulaması

Bu proje, Flutter, MongoDB ve Node.js kullanarak geliştirilmiş bir mobil kafe sipariş uygulamasıdır. Kullanıcıların kafe menüsünü görüntüleyip sipariş verebilmelerini sağlar.

## Özellikler

- **Kullanıcı Kaydı ve Girişi**: Kullanıcılar uygulamaya kayıt olabilir ve giriş yapabilir.
- **Menü Görüntüleme**: Kullanıcılar kafenin menüsünü görüntüleyebilir.
- **Sipariş Verme**: Kullanıcılar seçtikleri ürünleri sepete ekleyip sipariş verebilir.
- **Ödeme Yapma**: Kullanıcılar siparişlerini online olarak ödeyebilir.
- **Bildirimler**: Sipariş durumu hakkında bildirimler alabilirler.

## Teknolojiler

- **Flutter**: Mobil uygulama geliştirme için kullanıldı.
- **MongoDB**: Veritabanı yönetimi için kullanıldı.
- **Node.js**: Sunucu tarafı geliştirme için kullanıldı.

## Kurulum

1. **Depoyu Klonlayın**:
    ```bash
    git clone https://github.com/ayselcetin07/my_cafe_app.git
    ```
2. **Bağımlılıkları Yükleyin**:
    ```bash
    cd my_cafe_app
    flutter pub get
    npm install
    ```
3. **Veritabanı Bağlantısını Ayarlayın**:
    - `config.js` dosyasını açın ve MongoDB bağlantı bilgilerini girin.

4. **Uygulamayı Çalıştırın**:
    - Flutter uygulamasını çalıştırmak için:
        ```bash
        flutter run
        ```
    - Node.js sunucusunu çalıştırmak için:
        ```bash
        npm start
        ```
